# Security Policy

## Supported Scope
This project is intended for locally authorized interoperability research and desktop automation of Verdent installed on your own machine.

## Reporting
If you find a vulnerability in this wrapper itself, open a private security advisory or share a sanitized report with reproduction steps, impact, and evidence.

## Sensitive Data Rules
- Do not publish real `api_token`, JWT, Keychain exports, or local user content.
- Sanitize database samples and SSE payloads before sharing.
- Do not commit extracted Verdent application code or proprietary assets.
