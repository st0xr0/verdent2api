# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-08
### Added
- Reverse-engineered local database access for Verdent projects, sessions, and messages.
- Sidecar token capture and nonce-derived `api_token` support.
- `chat_stream` session management, prompt forwarding, and control/raw message APIs.
- Aggregated views for merged messages, tool uses, subagent child runs, and SSE streaming.
- Reverse engineering notes and architecture documentation.
